## Project Title

AI future directions – Week 6: Edge AI & IoT Smart Agriculture System

### Project Description
This project implements a Smart Agriculture Simulation System integrating AI, IoT, and Edge Computing. The system simulates field sensor data, processes it via an AI model to predict crop yield, and provides actionable insights for farmers. It demonstrates:
Edge AI concepts using TensorFlow Lite
IoT data flow using simulated sensors
A predictive model for crop yield estimation
Cloud/Edge hybrid architecture suitable for real-world applications
This repository is designed for learning and demonstration purposes in the context of emerging AI technologies and real-time IoT applications.

### Features
Simulation of IoT sensor data (soil moisture, temperature, humidity, pH, etc.)
Preprocessing and cleaning of time-series agricultural data
Machine Learning model for crop yield prediction
TensorFlow Lite model conversion for Edge AI deployment
Data flow diagram illustrating IoT → Cloud → AI pipeline
Modular Python scripts for training and simulation
Extensible architecture for real-world sensor integration
